Overview

The HTML Help Update object contains files that are normally installed by Hhupd.exe.

Return to Contents

System requirements

This object requires Windows 95 or Windows NT 4.

NOTE: Windows 98, ME, 2000 and XP already have the files installed by this object.

Return to Contents

Dependencies

Microsoft Internet Explorer 4.01. See Q245477 and Q164539.

Return to Contents

What this object does

This object performs the following actions:

  1. Installs the files listed below.

  2. Registers the self-registering file hhctrl.ocx.

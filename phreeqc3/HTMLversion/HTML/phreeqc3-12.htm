END

This keyword has no associated data. It ends the data input for a simulation. After this keyword is read by the program, the calculations described by the input for the simulation are performed and the results printed. Additional simulations may follow in the input file, each in turn will be terminated with an END keyword or the end of the file.

Example problems

The keyword END is used in all example problems, 1 through 22.
